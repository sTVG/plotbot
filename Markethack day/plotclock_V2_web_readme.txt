
in the code, change the SSID / Password for your local Lan

After programming module, start serial monitor

the module will report connection to the LAN, and it's IP address

in a Browser connected to the LAN, enter the IP address

A config page will appear - 
all changes to calibration values are only saved on "EXIT CALIBRATION" "Calibration mode off"

The LED will change colour depending on Mode 
Red Flash = Calibration
Blue= Text
Green = Time
